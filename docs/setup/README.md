## Common Issues

All clients have similar setup — usually just:

1. Install OpenControl via `Install.ps1`
2. Edit config file (2-3 lines added)
3. Restart client
4. Test with "Take a screenshot"

**Issues?** See [TROUBLESHOOTING.md](../../TROUBLESHOOTING.md) for fixes.

## Custom MCP Hosts

Using a custom MCP host? Point it to:

```
C:\Program Files\OpenControl\OpenControl.exe
```
